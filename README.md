# chip-8

a small chip-8 interpreter with debugger


# what it do ?

it runs simple chip-8 roms

# dependencies

- SDL2(-dev)

# how do compile

## unix

```bash
    $ git clone git@github.com:fabulousduck/chip-8.git
    $ cd chip-8
    $ make
    $ ./chip-8 -f games/<game name>
```
